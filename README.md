# Form Tutorial App

シンプルな入力フォームアプリです。  
ユーザー情報（名前・メール・場所）を入力して一覧表示できます。

---

## 機能

- 入力フォーム（名前 / Email / 場所）
- 入力データの一覧表示
- フォーム送信時のバリデーション（未入力チェック）
- テーブル形式でのデータ表示

---

## 使用技術

- React
- useState（状態管理）
- JSX

---

## 画面イメージ

- 入力フォーム（テーブル形式）
- データ一覧テーブル

---

## 使い方

1. 名前・Email・場所を入力
2. 「追加」ボタンをクリック
3. 入力したデータが下の一覧に表示される

#  Habit Tracker

シンプルな習慣管理アプリです。 
日々の習慣を記録し、ストリーク（連続日数）を確認できます。

---

## 機能

- 習慣の追加
- 習慣の削除
- 完了チェック（チェックボックス）
- ストリーク（連続日数）の表示
- localStorageによるデータ保存(リロードしても消えない)

---

## 使用技術

- React
- useState（状態管理）
- useEffect（副作用）
- localStorage

---

## 画面イメージ

- 習慣入力フォーム
- 習慣一覧テーブル
- チェックボックスで完了管理

---

## 使い方

1. 習慣を入力
2. 「追加」ボタンをクリック
3. チェックボックスで今日の完了を記録
4. ストリークが自動で更新される

#  Movie app

シンプルな習慣管理アプリです。 
映画の名で検索し、検索された映画がある場合一覧で表示される。気になる映画を気になる一覧に追加、削除できます。

---

## 機能

- 映画の検索
- 一覧表示
- 気になる一覧に追加
- 気になる一覧に削除
- ローディング表示
- エラーハンドリング
- 外部APIから映画のデータを取得
- ポスター画像表示
- localStorageによるお気に入り保存

---

## 使用技術

- React
- useState（状態管理）
- useEffect（副作用）
- localStorage
- Fetch API

---

## 画面イメージ

- 映画名の検索
- [Search]ボタンまたはEnterキーで検索
- 気に入りボタン追加・削除
- 気に入り一覧表示
- 映画一覧表示

---

## 使い方

1. 映画名を入力
2. 「Search」をクリック
3. ⭐️ボタンで気に入りに登録

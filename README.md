#  Movie app

シンプルな習慣管理アプリです。 
映画の名で検索し、検索された映画がある場合一覧で表示される。気になる映画を気になる一覧に追加、削除できます。

---

## 機能

- 映画の検索
- 一覧表示
- 気になる一覧に追加
- 気になる一覧に削除
- ローディング表示
- エラーハンドリング
- 外部APIから映画のデータを取得
- ポスター画像表示
- localStorageによるお気に入り保存

---

## 使用技術

- React
- useState（状態管理）
- useEffect（副作用）
- localStorage
- Fetch API

---

## 画面イメージ

- 映画名の検索
- [Search]ボタンまたはEnterキーで検索
- 気に入りボタン追加・削除
- 気に入り一覧表示
- 映画一覧表示

---

## 使い方

1. 映画名を入力
2. 「Search」をクリック
3. ⭐️ボタンで気に入りに登録

---
